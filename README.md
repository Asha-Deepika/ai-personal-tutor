# AI Personal Tutor

 An advanced AI-powered study assistant that transforms raw study material into interactive learning tools like summaries, notes, flashcards, quizzes, and real-time analytics.

---

## Key Features

 **Smart Content Processing**
- Extracts clean text from PDFs and raw input
- Generates concise summaries using NLP models

 **AI-Powered Chat**
- Ask questions from your material or general concepts
- Integrated with OpenAI (ChatGPT) & Groq APIs

 **Automatic Notes & Definitions**
- Identifies key concepts and important sentences
- Extracts meaningful definitions using NLP

 **Flashcards Generator**
- Converts notes into interactive flashcards
- Helps in active recall and revision

 **Dynamic Quiz System**
- Auto-generates MCQs (Easy / Medium / Hard)
- Tracks performance and accuracy

 **Live Analytics Dashboard**
- Visualizes quiz performance using Plotly
- Includes real-time live chart updates

---
